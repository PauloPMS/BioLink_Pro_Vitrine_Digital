# BioLink_Pro_Vitrine_Digital
Cartão de Visitas Digital que reune em um único link canais de atendimento, desenvolvido com HTML5 e CSS3 puros, focado em responsividade, acessibilidade e boas práticas de UI/UX.

## 🚀 Demonstração

👉 Acesse o projeto online:  

[site_do_projeto](https://seu-usuario.github.io/seu-repositorio/)


## 🧠 Sobre o projeto

Esta é uma página de perfil pessoal com o objetivo de centralizar links importantes e apresentar informações profissionais de forma moderna e responsiva.

O projeto foi desenvolvido sem frameworks, utilizando apenas tecnologias fundamentais da web.

## 🎯 Funcionalidades

- Foto de perfil circular
- Nome e bio profissional
- Botões interativos para redes sociais
- Efeitos de hover nos botões
- Layout totalmente responsivo (mobile-first)
- Ícones SVG no rodapé
- Acessibilidade básica (alt e aria-label)

## 🛠️ Tecnologias utilizadas

- HTML5 semântico
- CSS3 (Flexbox + Box Model)
- Google Fonts (Poppins)
- SVG inline

## 📱 Responsividade

O layout foi desenvolvido para se adaptar a:

- Smartphones 📱
- Tablets 📟
- Desktops 💻

Sem quebra de layout ou scroll horizontal.

## ♿ Acessibilidade

- Uso de atributo `alt` em imagens
- Uso de `aria-label` em ícones
- Estrutura semântica com `header`, `main` e `footer`
